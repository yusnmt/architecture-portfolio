## 研究概要

敵対的生成ネットワークGANを用いた創造性を持つ人工知能、つまり建築設計・デザイン分野への応用を意図したCreative AIを作成し、Creative AIと設計者が共に作り上げていく設計プロセスを用いた建築計画を行った。

具体的には、計画物のブロックプラン、floor Plan、外観デザインに対してCreative AIを活用し、最終的な成果物として住宅の設計を行なった。
